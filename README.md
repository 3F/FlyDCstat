Поддержка:

* StrongDC-sqlite
* Любые другие клиенты на базе этих предков (совместимость не гарантированна)

http://flylinkdc.ru


**Использование:**

* Распаковать содержимое из архива в корневой каталог программы.
* Запустить двойным кликом (если по умолчанию файл проассоциирован с java), либо выполнив команду:
  
```
javaw -jar flyDCstat.jar ["path"]
```

path - *опционально для указания пути к файлу статистики, если утилита размещена не в корневом каталоге программы.*
  
* На системе должна быть развернута *JRE* - [Java Runtime Environment](http://www.java.com)

**Сборки**
